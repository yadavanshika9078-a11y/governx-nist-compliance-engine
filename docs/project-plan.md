# GovernX Development Plan

## Week 1: Foundation

- Define project scope and security checks
- Create mock cloud-security data
- Set up the Python backend structure
- Create a basic API endpoint

## Week 2: Compliance Scanner

- Read mock configuration data
- Detect failed security checks
- Store findings in a database
- Map findings to NIST CSF 2.0 categories

## Week 3: Risk and Dashboard

- Calculate compliance and risk scores
- Build a React dashboard
- Display findings, severity, and remediation actions

## Week 4: Testing and Reporting

- Add automated tests
- Generate a simple compliance report
- Improve documentation
- Record a short project demonstration
